# 米塔世界数字人导购终端拍摄任务书

## 拍摄目的

本次拍摄只围绕一个目标：让观众看懂米塔世界数字人导购终端如何在门店里帮助顾客。

每条片子都按同一条逻辑拍：

```text
顾客在商品前出现一个真实状态
终端识别这个状态
终端给出一句可执行的建议
顾客获得更自然的服务体验
```

真人店员可以出现，但只作为执行辅助。画面主角必须是数字人导购终端，而不是真人销售。

## 当天交付

先拍 4 个核心场景。每个场景交付一组素材。

| 每个场景交付 | 要求 |
|---|---|
| 1 条成片 | 16-25 秒，能看懂完整过程 |
| 1 条干净原片 | 无字幕、无花字，方便后续复用 |
| 1 条终端 UI 特写 | 看清终端显示的观察、判断、建议、话术 |
| 3-5 张关键帧 | 顾客状态、终端 UI、执行动作、顾客反应 |
| 1 份拍摄卡 | 记录本条怎么拍、是否合格 |

当天最小成果：

```text
4 条成片
4 条干净原片
4 条终端 UI 特写
16-20 张关键帧
1 份素材清单
```

## 单条片子结构

| 时间 | 拍什么 |
|---|---|
| 0-4 秒 | 顾客在商品前表现出一个状态：犹豫、试用、回避、接近柜台 |
| 4-7 秒 | 镜头带到米塔世界数字人导购终端 |
| 7-11 秒 | 终端屏幕显示识别结果和建议话术 |
| 11-16 秒 | 数字人说话，或店员按终端建议执行 |
| 16-25 秒 | 顾客出现反应：继续停留、回应、试用、离开 |

## 画面要求

主镜头尽量让顾客、商品和终端同框。至少要看清：

```text
顾客脸或侧脸
顾客视线
顾客手部动作
商品或价签区域
数字人终端屏幕
```

终端屏幕不要只显示欢迎语。每条都要有四层信息：

```text
观察：顾客正在做什么
判断：顾客可能需要什么
建议：现在应该怎么服务
话术：数字人可以怎么说
```

屏幕文案用自然中文，不使用技术代号。

## 四个必拍场景

### 场景 1：价格犹豫

适合商品：鞋、手表、珠宝、手机、家电。

顾客在两三件商品之间停留，反复看价签或活动牌，身体仍朝向商品，没有快速离开。这个状态要拍得清楚，不能只让顾客站着不动。

终端 UI：

```text
观察：顾客停留较久，反复查看价格和两个商品。
判断：顾客可能在比较价格和价值。
建议：先做价值对比，避免直接强推。
话术：我看您在比较这几款。如果愿意，我可以帮您把价格、材质和适用场景简单对比一下，您不用马上决定。
```

执行方式：数字人直接说话；如果店员辅助，店员站在顾客侧前方半步外，用手势指向两到三个商品做横向对比。

顾客反应：继续停留，视线回到商品，愿意听一个简短比较。

### 场景 2：主动试用

适合商品：手机、家电、鞋、美妆、笔记本电脑。

顾客正在试用、翻看、按按钮、试穿或查看细节。手部动作要清楚，视线要集中在商品上。

终端 UI：

```text
观察：顾客正在主动试用，并持续查看商品细节。
判断：顾客已有兴趣，正在确认使用体验。
建议：做一个短演示，只讲关键差异。
话术：我只演示一个最关键的地方，您看这个细节是不是符合您的使用习惯。
```

执行方式：数字人引导顾客看一个关键功能；如果店员辅助，只做 3 秒以内的短演示，不要长篇推销。

顾客反应：继续试用，视线更集中，手部重新触碰或操作商品。

### 场景 3：回避或准备离开

适合商品：美妆、珠宝、手表、鞋。

顾客身体转离商品，脚尖朝外，不看店员或终端，手从商品附近收回。表演要自然，不要演成生气。

终端 UI：

```text
观察：顾客视线离开商品，身体有离开倾向。
判断：当前互动意愿较低。
建议：先观察，不要追问或强推。
话术：您先慢慢看，需要的时候我再帮您。
```

执行方式：数字人低压力提示；如果店员辅助，只点头后退半步，不继续追问。

顾客反应：没有被进一步打扰，可以继续短暂停留，也可以自然离开。

### 场景 4：接近柜台

适合商品：手机、家电、珠宝、手表、美妆。

顾客主动走向柜台、服务台或终端，视线朝向服务区域，可能手持商品或停在咨询点前。

终端 UI：

```text
观察：顾客主动接近服务区域，可能准备咨询。
判断：顾客接近决策或需要一个明确服务入口。
建议：用开放式问题确认最关心的一点。
话术：您现在主要想先确认哪一点？我可以按一个点帮您看，不会耽误太久。
```

执行方式：数字人直接询问；如果店员辅助，店员站在侧前方，保留距离，用开放手势邀请顾客说需求。

顾客反应：给出一个需求点，继续停在柜台前，互动进入服务流程。

## 拍摄卡

每条开拍前，导演或场记按这个模板读一遍。

```text
clip_id:
场景:
商品:
顾客人设:
机位:

顾客前 4 秒怎么演:
终端屏幕显示什么:
数字人说什么:
店员是否辅助，怎么站、怎么做:
顾客最后怎么反应:

现场是否通过:
需要重拍的原因:
```

示例：

```text
clip_id: MEETA_PRICE_COMPARE_001
场景: 价格犹豫
商品: 鞋
顾客人设: 谨慎比较，关注性价比
机位: 顾客、商品、终端同框中景

顾客前 4 秒怎么演:
顾客在两双鞋之间来回看，短暂停在价签区域，手伸向其中一双又收回。

终端屏幕显示什么:
观察顾客停留较久，反复查看价格和两件商品；建议先做价值对比。

数字人说什么:
我看您在比较这几款。如果愿意，我可以帮您把价格、材质和适用场景简单对比一下，您不用马上决定。

店员是否辅助，怎么站、怎么做:
如需店员辅助，站侧前方半步外，指向两双鞋做横向比较，不挡住商品。

顾客最后怎么反应:
继续停留，视线在终端、商品和价签之间切换，手重新靠近其中一双鞋。
```

## 现场重拍标准

出现下面情况，直接补拍。

| 问题 | 判断标准 |
|---|---|
| 顾客状态不清 | 关掉声音也看不出顾客是在犹豫、试用、回避还是准备咨询 |
| 终端存在感弱 | 观众看不出建议来自米塔世界数字人导购终端 |
| UI 内容太空 | 屏幕只有欢迎语，没有观察、判断、建议、话术 |
| 店员抢戏 | 画面变成真人店员销售，数字人终端不再是核心 |
| 执行动作不清 | 看不出数字人或店员具体说了什么、做了什么 |
| 顾客反应不清 | 看不出顾客是继续停留、回应、试用、后撤还是离开 |
| 画面遮挡 | 顾客脸、视线、手、商品或终端屏幕被挡住 |
| 表演过度 | 像短剧或硬广，缺少真实门店感 |
