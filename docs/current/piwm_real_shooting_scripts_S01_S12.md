# PIWM 真实拍摄脚本全集 S01-S12

更新时间：2026-05-11 CST

本文件是唯一给拍摄团队使用的脚本文件。所有状态、A/B 版本、机位、终端响应、演员动作、顾客台词、后续反应和 QA 标准都写在本文内；现场不需要跳转到其他文档。

## 统一拍摄规则

- 每个状态拍 A/B 两条，每条 10 秒。
- A 版是较合适的终端响应，B 版是失当或对照响应。
- A/B 前 3 秒必须一致：同一位置、同一商品、同一顾客起始状态，只改变 3 秒后的终端响应和顾客后续反应。
- FPV 是训练主视角；S03、S05、S07、S11 必须同步拍 HERO 视角。
- 智能售货柜可以通过屏幕、语音、灯效、柜内动作表达响应，不使用真人导购站位作为主口径。
- B 版失当响应要克制真实，不能演成夸张冲突。
- 每条素材收工前检查：起始状态是否可见、终端响应是否清楚、后 5 秒顾客反应是否可见、A/B 前 3 秒是否一致、是否无敏感信息。

## 统一命名

```text
G<组号三位>_S<状态两位>_<A|B>
示例：G001_S05_A
```

## S01 路过无关注 · 拍摄脚本

clip_id：`G001_S01_A` 与 `G001_S01_B` | 时长：每条 10 秒 | 机位：FPV，HERO 可选

## 一句话场景

顾客只是从智能酒柜前经过，对柜内商品只有一次短暂扫视，没有形成停留。A 版售货柜保持完全静默，让顾客自然路过；B 版售货柜误判兴趣并强推单品，顾客被打扰后加速离开。

## 商品

| 仓位 | SKU | 描述 | 价签 |
|---|---|---|---|
| 左 | SKU_X | 酱香型白酒 500ml 礼盒装 | ¥298 |
| 右 | SKU_Y | 浓香型白酒 500ml 简装 | ¥268 |

## 时间表 · A 版（合适响应：暂不打扰 - 完全静默）

| 秒数 | 演员 / 顾客动作 | 顾客自然台词 | 售货柜语音 / 宣发画面备注 |
|---|---|---|---|
| 0-1 | 从画面左侧或正前方自然走入，脚步不停 | - | 待机画面，柔和呼吸光 |
| 1-2 | 眼神短暂扫过柜内商品，不转肩 | - | 不弹卡、不出声 |
| 2-3 | 继续向出口方向走，身体主方向不朝柜子 | - | 不介入 |
| 3-5 | 经过柜前，步伐保持自然 | - | `Hold(mode=silent)`，屏幕极简待机 |
| 5-10 | 自然离开画面，不回头 | - | 维持静默 |

反应类型：自然路过 / 不被打扰 / 低意图保持低干扰。

## 时间表 · B 版（失当响应：建议推荐 - 力度强势）

| 秒数 | 演员 / 顾客动作 | 顾客自然台词 | 售货柜语音 / 宣发画面备注 |
|---|---|---|---|
| 0-3 | 与 A 版完全相同：短暂扫视但不停步 | - | 待机画面 |
| 3-4 | 听到声音后脚步一顿，眼神避开屏幕 | - | 单品全屏卡 + “立即购买” |
| 4-5 | 肩膀轻微收紧，步伐变快 | 「不用了。」 | 「这款最适合您，建议直接选这款。」 |
| 5-7 | 加速离开柜前，身体转向出口 | - | 强引导色维持 |
| 7-10 | 不再看商品，离开画面 | - | 屏幕维持推荐卡 |

反应类型：防御加速离开 / 被打扰 / 低意图被误判。

## A/B 对照关键

- 前 3 秒必须同一路径、同一速度、同一次短暂扫视。
- A 版没有任何售货柜存在感；B 版只在 3 秒后开始强推。
- 禁止演员在 A 版停下看商品，否则 S01 状态不成立。

## QA 信号

| 轴 | 必须可见 |
|---|---|
| 视线 | 只短暂扫过商品，不形成锁定 |
| 身体 | 行走方向不改变，不朝柜子站定 |
| 手部 | 不触玻璃、不指商品、不操作屏幕 |
| 反应 | B 版被打扰后明显加速离开 |

顾客刚进入商品区，还没有形成明确目标。A 版售货柜只做低压提示，让顾客愿意继续靠近；B 版立刻强推，顾客产生防御并后撤。

- A 版只提供“可选信息”，不能追问预算或推荐单品。
- B 版要在顾客还没停稳时介入，突出时机错误。

| 手部 | 全程不拿商品，避免和 S07 混淆 |
| 语音 | A 版接受低压提示；B 版表达“还没想买” |

顾客已经完成购买，手持从取货口拿出的商品准备离开。A 版售货柜礼貌告别，体验闭合；B 版售货柜购后仍强推另一款，顾客觉得被打扰。

## 时间表 · A 版（合适响应：招呼问候 - 告别）

| 秒数 | 演员 / 顾客动作 | 顾客自然台词 | 售货柜语音 / 宣发画面备注 |
|---|---|---|---|
| 0-1 | 从取货口方向拿起商品，身体转向出口 | - | 订单完成页 |
| 1-2 | 看一眼手中商品或小票 | - | 取货口指示灯结束 |
| 2-3 | 准备离开，表情平稳 | 「好了。」 | 订单完成页 |
| 3-4 | 听到告别后自然点头 | - | 简洁感谢页 |
| 4-5 | 转身离开 | 「谢谢。」 | 「感谢您的光临，期待下次再见。」 |
| 5-10 | 手持商品自然走出画面 | - | 柔和呼吸光，回待机 |

反应类型：自然离开 / 体验闭合 / 购后不再干扰。

## 时间表 · B 版（失当响应：建议推荐 - 力度强势）

| 秒数 | 演员 / 顾客动作 | 顾客自然台词 | 售货柜语音 / 宣发画面备注 |
|---|---|---|---|
| 0-3 | 与 A 版完全相同：已拿到商品，准备离开 | 「好了。」 | 订单完成页 |
| 3-4 | 屏幕突然推荐另一款，顾客停顿 | - | 单品推荐卡 |
| 4-5 | 皱眉，看向屏幕后移开视线 | 「我都买完了。」 | 「这款也很适合您，建议一起带走。」 |
| 5-7 | 把商品拿稳，直接转身 | - | 强引导色 |
| 7-10 | 离开更快，不再回应 | 「不用了。」 | 推荐卡维持 |

反应类型：购后被打扰 / 体验变差 / 快速结束互动。

## QA 信号

| 轴 | 必须可见 |
|---|---|
| 购买证据 | 手持取货口商品或从取货口拿出商品 |
| 身体 | 朝出口离开 |
| 手部 | 持商品自然离开，不空手假装 |
| 反应 | A 版自然闭合；B 版明确不满或回避 |
