# Batch 001 Seed

Purpose: document the existing 72 runtime rules as a seed corpus.

This is not a true source-to-rule distillation batch. The rules were mirrored
from `piwm_data/rules.py` by `_seed_generator.py` so the project has an
auditable baseline. Later batches should add real source-backed principles and
then revise, retain, or remove seed rules through review.

Promotion policy:

- Do not promote anything from this batch.
- Use this batch only as the provenance anchor for current v2.1 behavior.
- v2.2 changes should be introduced through `batch_002` and later.
