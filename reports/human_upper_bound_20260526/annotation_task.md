# Human Upper Bound 标注任务说明

## 标注目标

你是一位智能售货机的导购决策者。请看每条样本中的 3 帧顾客视频画面和顾客状态描述，从候选介入动作中选择你认为最合适的动作类型。

## 5 个动作含义

- **Greet**: 主动招呼，例如顾客刚走近时表达欢迎。
- **Elicit**: 主动询问需求，使用开放问题，不直接推销。
- **Inform**: 主动提供信息，帮助顾客理解商品或差异。
- **Recommend**: 主动推荐产品，给出明确或轻量的推荐。
- **Hold**: 等一等，不介入，让顾客自己看。

## 标注原则

- 候选动作里，Hold 表示“不介入，保持安静或低打扰”。
- 不要过度介入；如果顾客自己在看且没有困惑或求助信号，Hold 是合理选择。
- 没有“标准答案”给 annotator；请选择你认为最合理的动作。
- 每条样本预计 1-2 分钟，30 条总计约 30-60 分钟。

## 填写方式

请在 `annotation_template.csv` 中填写：

- `sample_id`: 已预填，不要修改。
- `chosen_action`: 填动作类型，只能是 `Greet` / `Elicit` / `Inform` / `Recommend` / `Hold` 之一。
- `notes`: 可选，写下犹豫点或理由。

每个 sample markdown 中的动作 ID 仅供核对候选动作；最终评分按 5-act 动作类型计算。
